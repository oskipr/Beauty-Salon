# Beauty Salon PR – Landing Page

Landing page para **Beauty Salon PR** en Viejo San Juan, Puerto Rico.  
Construida con Vite, React, TypeScript, Tailwind CSS y componentes de shadcn-ui.

## Scripts

- `npm run dev` – servidor de desarrollo en `http://localhost:8080/`
- `npm run build` – build de producción
- `npm run preview` – previsualizar el build
- `npm run lint` – ejecutar ESLint
- `npm run test` – ejecutar tests con Vitest

## Desarrollo local

```sh
npm install
npm run dev
```

Luego abre `http://localhost:8080/` en tu navegador.
